Realny projekt własny

Marketingowa strona AISoftware Studio

To własny projekt techniczny, a nie wdrożenie klienta.

Kontekst i problem

Potrzebowałem czytelnej strony, która rozdziela demo, development i kontakt bez udawania gotowego produktu klienta.

Decyzje i architektura

Strona korzysta z Angulara na froncie oraz formularza połączonego z API FastAPI. Frontend i backend pozostają osobnymi aplikacjami.

Rezultat techniczny

Projekt obejmuje typowane treści, testy, automatyzację jakości oraz konfigurację wdrożenia na Google Cloud.

Ograniczenia i kolejne kroki

Nie opisuje ruchu, konwersji ani wyników klientów. Kolejne prace zależą od potwierdzonego zakresu i potrzeb użytkowników.
