# Protolume brand assets

The optional logo assets are expected to use these names:

- `protolume-logo-horizontal-dark.svg`
- `protolume-logo-horizontal-light.svg`
- `protolume-symbol.svg`
- `protolume-symbol-mono.svg`
- `favicon.svg`
- `social-preview-1200x630.png`

Logo SVGs should use a transparent background and provide the dark, light,
symbol, and monochrome variants described by their filenames. The social
preview must be a PNG at 1200x630. Do not use a large JPG as a header logo.

The shared logo component intentionally uses a text fallback until these
assets are supplied and configured. No placeholder SVGs are included.
